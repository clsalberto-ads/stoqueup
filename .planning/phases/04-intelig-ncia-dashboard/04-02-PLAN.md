# Plan: 04-02-ANALYTICS

Implementação das queries complexas para cálculo de médias móveis e indicadores de risco.

## Tasks

<task id="1" type="execute">
<read_first>
- src/db/schema.ts
- src/lib/inventory-actions.ts
</read_first>
<action>
Criar o arquivo `src/lib/analytics-actions.ts`.
Implementar a função `getProductMetrics(productId: string)`.
A função deve usar `sql` do Drizzle para:
1. Gerar uma série de datas (últimos 30 dias).
2. Fazer join com `inventoryLogs` (tipo SALE).
3. Calcular a média móvel de 7 e 30 dias.
4. Retornar: `averageSales7d`, `averageSales30d`, `daysOfStockRemaining` (baseado no currentStock).
</action>
<acceptance_criteria>
- `getProductMetrics` retorna os valores calculados corretamente.
- Trata casos de média zero (evita divisão por zero).
</acceptance_criteria>
</task>
